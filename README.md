Para rodar este trabalho é necessário ter o gerenciador de pacotes Yarn e Node.js instalados.

Para rodar o servidor, entre na pasta server no terminal e digite:

- yarn (para instalar dependencias)
- yarn start (para rodar o servidor)

Para rodar o cliente, entre na pasta client no terminal e digite:

- yarn (para instalar dependencias)
- yarn dev (para rodar o client)

Depois disso entre na url que aparece no terminal do cliente e abra em dois navegadores.
Eles devem poder jogar Damas entre si.
